🎉 LATEST PUBLICATION

On the Identifiability of Genetic Parameters for Growth in Mice With a Massively Deep Pedigree

Xi Ding, A. A. Musa, Norbert Reinsch

Animal Breeding and Genetics, 2025

🧬

📋 Research Significance

This groundbreaking study addresses critical challenges in quantitative genetics by investigating genetic parameter identifiability in massively deep pedigree structures. Using a unique mouse population spanning over 40 generations, we developed novel diagnostic methods for detecting and resolving parameter estimation issues in complex breeding populations.

🔬 Key Findings

  • Identified specific conditions under which genetic parameters become non-identifiable
  • Developed computational methods for detecting identifiability issues
  • Provided practical guidelines for breeding program optimization
  • Demonstrated methodology using 50,000+ animals across 40+ generations

🌟 Impact

This research provides breeding programs worldwide with essential tools for ensuring reliable genetic evaluations, leading to more accurate selection decisions and improved breeding efficiency.

📊 Technical Details

Mixed Models BLUP R Programming ASReml

Statistical analysis using advanced mixed linear models with custom R implementations for identifiability testing.

🛠️ COLLABORATIVE WORK

CLARITY: a Shiny app for interactive visualisation of the bovine physical-genetic map

Nina Melzer, Saber Qanbari, Xi Ding, Dörte Wittenburg

Frontiers in Genetics, 2023

🗺️

🎯 Project Overview

CLARITY is an innovative R Shiny application that provides researchers with an interactive platform to explore bovine genetic and physical maps. This tool democratizes access to complex genomic visualization, making it easier for researchers worldwide to conduct fine-mapping studies and genomic research.

⚙️ My Contribution

  • Data processing and quality control algorithms
  • Interactive visualization components development
  • User interface design and testing
  • Documentation and user guide preparation

🌍 Global Impact

CLARITY has been adopted by researchers worldwide for bovine genomic studies, providing free access to sophisticated genomic visualization tools that were previously available only to well-funded laboratories.

Full-stack Shiny application with advanced data visualization capabilities.
